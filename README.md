# BlockFiller

一个用于快速填充区域的简单 Minecraft 插件。

---

## 📌 功能说明

* 左键选择第一个点
* 右键选择第二个点
* 蹲下 + 右键开始执行填充
* **填充方块来源：第一个选点上方的方块**

---

## 🎮 使用方法

1. 使用指令获取工具：

   ```
   /blockfiller
   ```

2. 手持工具：

   * 左键点击方块 → 设定 **点1**
   * 右键点击方块 → 设定 **点2**

3. 选择填充方块：

   * 插件会自动读取 **点1上方的方块类型** 作为填充材料

4. 执行填充：

   * **蹲下 + 右键** → 执行填充操作

---

## 🔐 权限

| 权限节点              | 说明        |
| ----------------- | --------- |
| `blockfiller.use` | 使用插件及获取工具 |

---

## ⚙️ 注意事项

* 必须先选定两个点才能执行
* 填充方块由 **点1上方方块决定**，请提前放置好
* 大范围操作可能对服务器性能产生影响

---

## 📄 说明

本插件适用于需要快速进行区域填充的场景，如建筑、地图制作等。
